Create `src/app/api/og/route.tsx` using Next.js built-in `next/og` ImageResponse (no npm install needed -- it ships with Next.js 16).

```typescript
import { ImageResponse } from 'next/og'
import { NextRequest } from 'next/server'
```

**Route handler:**
- Export `GET` function accepting NextRequest
- Extract query params: `title` (required), `subtitle` (optional), `category` (optional)
- If no title provided, return a default fallback image with "Richard Hudson | Revenue Operations"
- Return `new ImageResponse(jsx, { width: 1200, height: 630 })`

**Design (branded, clean, professional):**
- Background: white (#fafafa) with a subtle gradient accent bar at top (use brand colors -- dark navy #1a1a2e to accent blue #4a90d9)
- Top accent bar: 8px tall, full width, linear gradient left-to-right #1a1a2e to #4a90d9
- Layout: flex column, centered vertically and horizontally, padding 60px
- Title: 48px font weight 700, color #1a1a2e, max 2 lines (truncate with ellipsis if needed)
- Subtitle/category: 24px font weight 400, color #6b7280, shown below title if provided
- Bottom: "richardwhudsonjr.com" in 20px, color #9ca3af
- Author line: "Richard Hudson" in 28px font weight 600, color #374151, positioned between title and URL

**Edge runtime:**
- Add `export const runtime = 'edge'` for Vercel edge deployment
- No external font fetching needed -- use system fonts (the ImageResponse uses satori which supports system fonts)

**Cache headers:**
- Set `Cache-Control: public, immutable, max-age=31536000` for aggressive caching (OG images are deterministic based on query params)

**Input validation:**
- Title max length: 120 chars (truncate if longer)
- Subtitle max length: 80 chars
- Category max length: 40 chars
- All params URL-decoded automatically by NextRequest searchParams
